Script for sorting exam scores. Or any other scores.

Usage:
run script and follow the orders.

<pre>
python sort.py 
Input file name: results.txt
index column number: 2
score column number: 3
</pre>

example input file:
<pre>
1. 123456 15
2. 123457 13
3. 123458 16
4. 123459 12
5. 123460 10
6. 123461 8
</pre>
